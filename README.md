# Unified UPI Autopay Tracker & Manager

## Problem
Users forget where they enabled autopay across UPI apps like GPay, PhonePe, and Paytm, making it hard to track and cancel subscriptions.

## Solution
A unified platform that detects recurring payments and shows all autopay subscriptions in one place.

## Features
- Detect recurring payments
- Show subscriptions dashboard
- Monthly expense tracking
- Autopay management insights

## Tech Stack
- Frontend: React
- Backend: FastAPI
- Database: PostgreSQL
